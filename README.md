# 📂 Лучший проект

## 🍔 Product Analytics of a Food Delivery Mobile App

Комплексный проект по продуктовой аналитике мобильного приложения сервиса доставки еды.

### Что сделано

- Очистка и подготовка данных
- Исследовательский анализ данных (EDA)
- Расчет продуктовых метрик (DAU, WAU, MAU, Stickiness)
- Анализ монетизации (AOV, ARPU, ARPPU)
- Анализ пользовательской конверсии
- Построение воронки пользовательских событий
- Анализ потерь пользователей между этапами воронки
- Визуализация ключевых продуктовых показателей

### Технологии

Python • Pandas • NumPy • Matplotlib • Seaborn • Jupyter Notebook
